1)Открыть скрипт создания готовой базы данных RegistraciaDB. Скрипт откроется в MSQL Server Management Studio в виде запроса.
2)Выполнить запрос. Будет созданная готовая база данных.
3)Открыть проект Registration.sln и проверить подключение БД к проекту, если подключение отсутсвует необходимо его создать.
4)В обозревателе решений найти строку App.config и открыть.  Найти строку подключения
  <connectionStrings>
    <add name="Entities" .../>
  </connectionStrings>
  
5)Если параметр add name не совпадает с приведеным выше, то переименовать в Entities.
